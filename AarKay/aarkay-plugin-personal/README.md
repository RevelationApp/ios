# aarkay-plugin-personal

[![AarKay](https://img.shields.io/badge/RahulKatariya-AarKay-red.svg)](http://github.com/RahulKatariya/AarKay)

Personal AarKay Plugin

Installation

- Add https://github.com/RahulKatariya/aarkay-plugin-personal to your `~/AarKay/AarKayFile`
- Run `aarkay update`.
- That's it, You can start using the Personal.
